## 🛠️ Summer 2025 Project Breakdowns

### Assignment 1: Foundational Python Programming
* **Topic:** Core Programming Concepts
* **Description:** Efficient implementation of foundational syntax, structural data built-ins (lists, dictionaries, tuples, sets), and optimized control loops.

### Assignment 2: Object-Oriented Programming (OOP) in Python
* **Topic:** OOP Architecture
* **Description:** Designing modular and modular software patterns including clean implementation of classes, objects, inheritance, polymorphism, and encapsulation.

### Assignment 3: Data Visualization
* **Topic:** Graphical Data Interpretation
* **Datasets:** Gas Prices Dataset, FIFA Football Dataset
* **Description:** Transforming complex multivariate data into actionable charts using visualization libraries to identify latent patterns and tracking distributions.

### Assignment 4: Exploratory Data Analysis (EDA)
* **Topic:** Deep Structural & Statistical Analysis
* **Dataset:** Titanic Dataset
* **GitHub Repository:** [Assignment-4](https://github.com/shrav-jally/Assignment-4)
* **Core Focus:**
  * Imputation of missing structural features.
  * Deep-dive univariate and bivariate feature distributions.
  * Correlation mapping via interactive visualization to pinpoint data dependencies.

### Assignment 5: Data Preprocessing & Feature Engineering
* **Topic:** Production Data Wrangling
* **Dataset:** House Prices Dataset (Advanced Regression Techniques)
* **GitHub Repository:** [Assignment-5](https://github.com/shrav-jally/Assignment-5)
* **Core Focus:**
  * Handling duplicate entries, missing attributes, and invalid structures.
  * Categorical feature encoding and numerical scale standardization.

### Assignment 6: Model Evaluation & Hyperparameter Tuning
* **Topic:** Predictive Optimization
* **Dataset:** Iris Dataset
* **GitHub Repository:** [Assignment-6.git](https://github.com/shrav-jally/Assignment-6.git)
* **Core Focus:**
  * Comparative testing across 3 unique ML model architectures.
  * Implementation of `GridSearchCV` and `RandomizedSearchCV` routines.
  * Validation mapping across performance matrices (Accuracy, Precision, Recall, $F_1$-score).

### Assignment 7: Machine Learning Model Deployment with Streamlit
* **Topic:** Web Service Architecture
* **Live Deployment:** [Streamlit Application Dashboard](https://lnkd.in/gH8Hbjpm)
* **GitHub Repository:** [Assignment-7](https://github.com/shrav-jally/Assignment-7)
* **Core Focus:**
  * Converting a serialized predictive model into a lightweight, browser-accessible user dashboard.
  * On-the-fly user query inference paired with downstream performance visuals.

### Assignment 8: RAG Q&A Intelligent Chatbot
* **Topic:** Generative AI & Semantic Document Search
* **Dataset/Context:** Loan Approval Data & Knowledge Corpus
* **GitHub Repository:** [Assignment-8](https://github.com/shrav-jally/Assignment-8)
* **Core Focus:**
  * Constructing a Retrieval-Augmented Generation context loop using localized vector embeddings.
  * Standardizing conversational orchestration logic via frameworks like `LangChain`.
  * LLM generation mapping leveraging lightweight open-weight models or specialized APIs.
